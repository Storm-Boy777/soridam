# AI 튜터링 커리큘럼 방향 — 강사님 의견 요청 메일

---

**제목**: [하루오픽] AI 튜터링 커리큘럼 방향에 대해 강사님 의견을 여쭙고 싶습니다

---

안녕하세요, 강사님.

하루오픽 개발을 맡고 있는 OOO입니다.
항상 베타 테스트에 소중한 피드백 주셔서 감사합니다.

현재 AI 튜터링 모듈을 설계하고 있는데, 방향을 잡기 전에 현직에서 학생들을 직접 가르치시는 강사님의 의견을 꼭 듣고 싶어서 메일 드립니다.

---

### 저희가 고민하고 있는 것

모의고사를 여러 번 보고 나면, 학생들이 "내가 뭘 고쳐야 등급이 오르는지" 모르는 경우가 많더라고요. 그래서 AI 튜터링으로 **구체적으로 뭘 연습해야 하는지** 알려주고, **직접 말하기 연습**까지 시키는 시스템을 만들려고 합니다.

---

### 설계 방향: "채점 기준을 하나씩 충족시키는 커리큘럼"

저희가 참고한 접근법은 이렇습니다:

**핵심 아이디어**: OPIc 채점관이 체크하는 항목을 순서대로 하나씩 충족시키자.

구체적으로는 이런 순서입니다:

**1단계. 문단 구조 (서론-본론-결론)**
- **모든 답변**(묘사, 루틴, 비교, 과거경험, 롤플레이, 어드밴스)에서 "문단으로 말하기"를 먼저 훈련
- 문장을 나열하는 게 아니라, 서론(주제 선언) → 본론(세 가지로 나눠서) → 결론(마무리) 구조를 잡는 것
- description 유형(각 콤보의 첫 문항)이 가장 쉽게 문단을 만들 수 있어서 연습 시작점으로 사용
- OPIc 세부진단서의 F2(Paragraph Length) 항목에서 "Skeletal paragraphs" 수준을 모든 답변에서 일관되게 유지해야 IH/AL 심사 대상이 된다는 관점

**2단계. 롤플레이 (질문하기 + 문제해결)**
- 11번에서 연속적이고 복잡한 질문을 충분히 할 수 있는지
- 12번에서 상황 설명 + 해결책을 구체적으로 제시할 수 있는지
- 이게 IM3 확보 → IH 심사 자격의 관문이라는 관점

**3단계. 시제 정확도**
- 과거 문제(어릴적/최근 활동/기억에 남는 경험)에서 시제를 정확하게 쓰는지
- "When I was young, I used to ~ / Unlike now ~ / As for now ~" 같은 틀을 주고, 주제만 바꿔 끼우는 방식
- 시제 정확도가 IH↔AL을 가른다는 관점

**4단계. 공통형 주제(서베이 미선택 주제) + 어휘 업그레이드**
- 고르지 않은 주제에 대해서도 문단으로 답변할 수 있는지
- "many"를 "numerous"로, "good"을 "sophisticated"로 — 같은 내용인데 어휘만 1~2개 고급으로 바꾸면 등급이 오른다는 관점
- 학생의 답변을 AI가 AL 수준으로 교정한 버전을 만들어서, 원본과 비교해서 "여기가 달라요, 이 단어를 써보세요" 하는 방식

**5단계. 14번/15번 (비교변화 + 이슈)**
- 옛날과 지금의 차이 → "스마트폰 덕분에 다 편해졌다"로 대부분 커버
- 사회 이슈 → "경제/물가/인플레이션"으로 대부분 커버
- AL을 주기 위한 전용 문항이므로, 토론 구조 + 고급 어휘가 핵심

---

### 훈련 방식: "자유 답변 → 교정 → 다시 말하기"

모든 훈련은 이런 사이클로 진행합니다:

```
1. AI가 실전과 같은 질문을 제시
2. 학생이 자유롭게 답변 (녹음)
3. AI가 분석해서 피드백
   - 구조가 빠졌는지 (서론-본론-결론)
   - 문법 실수가 있는지 (수일치, 전치사 등)
   - 어휘가 너무 단순한지 (AL 대체 어휘 제안)
4. 학생이 피드백 반영해서 다시 말하기
```

핵심 원칙은 **"답변을 통째로 외우게 하지 않는다"**는 것입니다.
틀(구조)만 익히고, 내용은 학생이 자기 이야기로 자유롭게 채우는 방식입니다.

### 강사님께 여쭙고 싶은 것

실제로 학생들을 가르치시면서 느끼시는 관점에서:

1. **이 순서가 맞는 것 같으신가요?**
   - 문단 구조 → 롤플레이 → 시제 → 돌발/어휘 → 14/15번
   - 실제 수업에서도 이런 순서로 가르치시나요, 아니면 다른 순서가 더 효과적인가요?

2. **"문단 구조(서론-본론-결론)를 모든 답변에서 일관되게 유지해야 한다"는 관점에 동의하시나요?**
   - description뿐 아니라 routine, 과거경험, 심지어 롤플레이까지 문단 구조가 필요하다는 관점인데, 실제로 그런 차이를 보셨는지 궁금합니다.
   - OPIc 세부진단서 F2(Paragraph Length)에서 "Skeletal paragraphs" 체크가 IH/AL 여부를 가른다고 보시나요?

3. **어휘 업그레이드가 IH→AL에서 정말 효과가 있나요?**
   - many→numerous, good→sophisticated 같은 1~2개 어휘 교체만으로 등급이 바뀌는 경우가 실제로 있는지
   - 아니면 어휘보다 다른 요소(발음, 유창성, 내용 깊이 등)가 더 중요한지

4. **"만능 답변" 전략에 대해 어떻게 생각하시나요?**
   - 14번은 "스마트폰으로 다 편해졌다", 15번은 "경제/물가 이슈"로 어떤 주제든 대응하는 방식
   - 실제로 이런 전략이 통하는지, 아니면 채점관이 패턴을 간파해서 감점하는지

5. **등급별로 가르치는 방식이 달라야 할까요?**
   - 저희가 고민하는 부분인데, IM1 목표인 학생과 AL 목표인 학생에게 같은 커리큘럼을 쓸 수는 없을 것 같습니다.
   - 예를 들어 IM1 학생에게 "서론-본론(3가지)-결론" 구조를 요구하면 과부하가 될 것 같고, "문장 2~3개 연결하기"부터 시작해야 할 것 같은데 강사님도 그렇게 느끼시나요?
   - 반대로 IH 학생에게는 "many를 numerous로 바꾸세요" 같은 어휘 업그레이드가 효과적인지도 궁금합니다.

6. **AI 튜터링에서 가장 중요하게 다뤄야 할 것이 뭐라고 생각하시나요?**
   - 위에 설명드린 것 외에, 강사님이 학생들을 가르치시면서 "이것만 고치면 바로 등급이 오르는데" 하고 느끼시는 것이 있으시면 알려주시면 정말 큰 도움이 됩니다.
   - 특히 등급대별로(IM1~2 / IM3 / IH / AL) 가장 임팩트가 큰 교정 포인트가 다른지 궁금합니다.
