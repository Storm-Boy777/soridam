## 2. 개발자의 철학

### 왜 만들었는가

이 서비스는 **수익을 추구하지 않습니다.**

OPIc을 준비하면서 느꼈습니다. 기존 학습 서비스들은 비싸고, 본질을 놓치고 있다고. OPIc은 외우는 시험이 아닙니다. 자기 이야기를 자기 목소리로 하는 시험입니다. 그런데 대부분의 서비스는 스크립트를 외우게 합니다.

**나의 기술로 만든 이 도구가, 진짜 준비하는 사람들에게 도움이 되었으면 합니다.**

### 운영 원칙

- **비영리**: 수익을 가져가지 않습니다
- **종량제**: 사용한 만큼만 과금됩니다 (AI API 호출 비용)
- **기부 운영**: 모든 수익은 서버 유지비로만 사용됩니다
- **오픈**: 누구나 무료로 시작할 수 있습니다

### 소리담이 믿는 것

> "OPIc은 외우는 시험이 아니라, 자기 이야기를 말하는 시험이다."

- 당신의 평범한 일상이 가장 완벽한 답변입니다
- 시험의 구조를 알면, 그것이 곧 전략이 됩니다
- AI는 당신을 대체하는 것이 아니라, 당신의 말하기를 도와주는 도구입니다

---

## 3. 타겟 사용자

- OPIc 시험이 무엇인지 **이미 아는 사람들**
- 취업/승진/졸업을 위해 OPIc 점수가 필요한 20~30대
- 기존 학습법(암기)에 한계를 느낀 사람들
- 비용 부담 없이 체계적으로 준비하고 싶은 사람들

**→ "OPIc이 뭔지 설명할 필요 없음. 바로 도구를 제공."**

## 4. 핵심 기능 (6개 모듈)

| 모듈 | 설명 |
|------|------|
| **후기 분석** | 기출 빈도를 분석하고 출제 패턴을 파악합니다 |
| **스크립트** | 내 경험 기반 맞춤 스크립트를 AI가 생성합니다 |
| **모의고사** | 실전과 동일한 환경에서 74개 항목을 평가합니다 |
| **튜터링** | 취약점을 진단하고 집중 훈련을 처방합니다 |
| **쉐도잉** | 원어민 음성을 따라 읽으며 발음을 교정합니다 |
| **만능패턴** | 어떤 질문에도 쓸 수 있는 범용 표현을 훈련합니다 |

- **전문적**: 장난스럽지 않고, 도구다운 느낌
- **따뜻함**: 차갑지 않게. 블루지만 웜 크림 배경으로 중화

## 6. 랜딩 페이지 구성 (요청)

### 섹션 1: Hero
- "나의 목소리에 나의 이야기를 담다" (서브)
- **"말하다, 나답게."** (메인 헤드카피)
- "시작하기" CTA 버튼 1개
- 배경: 블루 글로우, 미니멀. 여백 적당히 (과도하게 비우지 말 것)

### 섹션 2: OPIc 전략 훅
- "OPIc" 큰 글자
- 타이핑 애니메이션: "어떤 시험일까요?"
- "많은 응시자들이 실제 어떤 시험인지 잘 모르고 있어요."
- **"어떤 시험인지 알면, 곧 전략이 됩니다."**
- "전략 가이드 보기 →" 링크

### 섹션 3: 기능 소개
- 6개 모듈 카드 (3열 2행 그리드)
- 각 카드: Lucide 아이콘 + 모듈명 + 한 줄 설명
- 클릭 시 해당 페이지로 이동
- 호버 시 아이콘 색 반전 (블루 배경 → 화이트 아이콘)

### 섹션 4: 운영 철학
- 하트 아이콘 (코랄)
- **"이 서비스는 수익을 추구하지 않습니다"**
- 설명 텍스트 3줄
- "기부로 운영되는 비영리 학습 도구입니다." (뮤트 컬러)

### 섹션 5: Final CTA
- "준비되셨나요?"
- **"지금 시작하세요"**
- "무료로 시작하기 →" 버튼
