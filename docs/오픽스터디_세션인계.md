### E. 멤버 홈 풀 리뉴얼 (`MemberHome.tsx` 신규)
**8가지 시간대 분기** (`SessionStateKind`):
1. **`live`** — 활성 세션 + 운영 시간 안 → joinedCount별 카피 ("지금 N명 함께 모여 있어요" / "한 분이 먼저 와 계세요" / "오늘 모임이 열렸어요")
2. **`live_overtime`** — 활성 세션 + 종료 후 grace 60분 이내 → "아직 함께 마무리하고 있어요"
3. **`ready`** — 활성 X + 시간 안 → "오늘 모임이 열렸어요" + "아직 아무도 입장하지 않았어요"
4. **`today_soon`** — 시작 30분 전 ~ 시작 → "곧 시작해요 · 모일 시간이에요" (입장 활성)
5. **`today_before`** — 30분 이상 전 → "오늘 {시간}에 모여요" (입장 비활성)
6. **`today_after`** — 종료 + grace 지남 → "오늘 모임은 끝났어요 · 다음에 또 만나요"
7. **`weekend`** — 운영일 X → "오늘은 쉬어가는 날이에요"
8. **`before_first`** — 첫 시작일 이전 → "첫 모임을 기다리고 있어요"

**상수**:
- `ENTRY_OPEN_MINUTES_BEFORE = 30` (시작 30분 전부터 입장 활성)
- `SESSION_GRACE_MINUTES = 60` (운영 종료 + 1시간 grace)

**기능**:
- 1초 카운트다운 (`getCurrentSessionState` 매 1초 호출)
- Realtime presence — 활성 세션 channel listen, 입장 멤버 추적
- 멤버 dot 시각 분리 (입장 컬러 vs 미입장 회색)
- 모바일 풀 너비 입장 버튼 + dot 우측 상단
- OnboardingGuide 운영 가이드 토글 (기본 펼침)
- 이번 주 일정 (운영일 ✓/●/○, 비운영일 작은 점)
- 학습 통계 (마지막 참여 / 누적 답변 / 시험 D-day) — 데이터 부족 시 ─
- 그룹 카드 + 빠른 이동 카드형
- ModeBadge (🌐 온라인 / 🏢 오프라인)

### G. lazy 정리 + lobby 의미 살림
- **`isSessionExpired(startedAtIso, schedule, now)`** — 세션 시작일의 운영 종료 + grace < 지금이면 만료
- **`getActiveSession`**: 만료된 활성 세션 자동 abandoned + null 반환
- **`createSession`**: 좀비 만료됐으면 자동 abandoned 후 새로 생성. **step DEFAULT('mode_select')** 사용 → lobby 단계로 시작
- **`endSession`**: 직접 종료 (사용자가 종료 버튼 클릭) — status='completed'
- **`abandonSession`**: 중도 이탈
- **`advanceLobby(sessionId)`** 신규 SA — lobby에서 "시작" 누르면 step만 mode_select → category_select로
- 멤버 홈 `handleEnter` — 활성 세션의 step에 따라 분기:
