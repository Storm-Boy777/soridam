### Step 1: 듣기 — `step-listen.tsx` (90줄)

**클라이언트 완결, 크레딧 없음**

- 전체 문장 목록 표시 (활성 문장 하이라이트)
- 표시 모드 토글: 영문 / 한글 / 둘 다
- 문장 클릭 → 해당 위치로 재생 이동 (seekRequest)
- 문장 반복 재생 토글 (toggleRepeat)
- 속도 조절: 0.75x / 1.0x / 1.25x / 1.5x

---

### Step 2: 따라읽기 — `step-shadow.tsx` (170줄)

**클라이언트 완결, 크레딧 없음**

- 문장별 순차 진행 (shadowIndex)
- 텍스트 힌트 3레벨 토글:
  - `full`: 전체 보기 (오버래핑 효과)
  - `first-word`: 첫단어 + 밑줄 (점진 난이도)
  - `hidden`: 음성만 (쉐도잉 효과)
- 진행도 바 (완료/전체 문장)
- 문장 끝 → 자동 완료 (shadowCompleted[] 추가)
- 이전/다음 문장 네비게이션

---

### Step 3: 혼자 말하기 — `step-recite.tsx` (153줄)

**클라이언트 완결, 크레딧 없음**

- 질문 텍스트만 표시 (음성 재생 없음)
- 타이머 표시 (경과 시간)
- MediaRecorder 녹음 + 재생
- **Peek 기능**: 막힐 때 다음 문장 영문 표시 (사용 횟수 추적)
- Zustand에 reciteTimer, recitePeekCount 저장

---

### Step 4: 실전 — `step-speak.tsx` (273줄)

**Edge Function 호출, 크레딧 1회 소비**

```
[실제 평가 흐름]

1. 크레딧 확인 (checkScriptCredit Server Action)
   └─ 부족 시 버튼 비활성화 + 경고 표시

2. 녹음 (MediaRecorder → Blob)
   └─ 타이머 표시

3. 세션 생성 (startShadowingSession Server Action)
   └─ shadowing_sessions INSERT → sessionId 반환

4. Base64 인코딩
   └─ Blob → ArrayBuffer → btoa()

5. Edge Function 호출 (클라이언트 직접 fetch)
   └─ POST /functions/v1/scripts/evaluate
   └─ body: { session_id, audio_base64, audio_duration }

6. 평가 결과 표시 (EvaluationResult 컴포넌트)
   └─ 캐시 무효화: script-credit, shadowing-history
```

**에러 처리:**
- 5단어 미만 발화 → 크레딧 환불 + "답변이 너무 짧습니다"
- API 오류 → 크레딧 환불 + 재시도 안내

### Edge Function — `scripts/index.ts` 내 evaluate 핸들러

**위치:** `supabase/functions/scripts/index.ts` (690~858줄)

```
[처리 순서]

1. 세션 조회 (session_id → user_id, script_id 확인)
2. 크레딧 차감 (consume_script_credit RPC)
   └─ 실패 → 402 응답
3. Whisper STT (model: whisper-1, language: en)
   └─ Base64 → Uint8Array → Blob(webm) → FormData
4. 유효성 검사 (5단어 미만 → 환불 + 400)
5. GPT-4.1 평가 (temperature: 0.3)
   └─ 입력: 질문, 스크립트, 핵심 표현, 목표 등급, 발화 텍스트
6. DB 저장
   └─ shadowing_sessions UPDATE (status='completed', audio_duration)
   └─ shadowing_evaluations INSERT (평가 결과 전체)
7. 응답 반환
```

### Zustand 스토어 — `lib/stores/shadowing.ts` (201줄)

```typescript
type ShadowingStep = 'listen' | 'shadow' | 'recite' | 'speak';
type TextHintLevel = 'full' | 'first-word' | 'hidden';

interface ShadowingState {
  // === 데이터 (패키지에서 로드) ===
  packageId: string | null;
  scriptId: string | null;
  sentences: TimestampItem[];
  audioUrl: string | null;
  questionText: string | null;
  questionKorean: string | null;
  keyExpressions: string[];

  // === 현재 단계 ===
  currentStep: ShadowingStep;

  // === Step 1: 듣기 ===
  isPlaying: boolean;
  currentTime: number;
  playbackRate: number;              // 0.75 | 1.0 | 1.25 | 1.5
  displayMode: 'both' | 'english' | 'korean';
  seekRequest: number | null;       // 문장 클릭 시 이동
  repeatTargetIndex: number | null;  // 문장 반복 재생

  // === Step 2: 따라읽기 ===
  shadowIndex: number;
  shadowHintLevel: TextHintLevel;
  shadowCompleted: number[];         // 완료된 문장 인덱스 배열

  // === Step 3: 혼자 말하기 ===
  reciteTimer: number;
  recitePeekCount: number;
  reciteShowPeek: boolean;

  // === Step 4: 실전 ===
  speakTimer: number;
  speakResult: ShadowingEvaluation | null;

  // === 공통 녹음 ===
  isRecording: boolean;
  recordingDuration: number;
}
```

**Persist (localStorage, key: `shadowing-progress`):**
- 저장: `packageId`, `scriptId`, `currentStep`, `shadowCompleted`, `shadowHintLevel`
- 저장 안 함: 실시간 상태 (isPlaying, currentTime, isRecording 등)

### 크레딧 연동

| Step | 크레딧 | 로직 |
|------|--------|------|
| 1~3 | 무료 | 클라이언트 완결, 체크 불필요 |
| 4 | **스크립트 크레딧 1회** | EF에서 `consume_script_credit` RPC 호출 |

**차감 순서:**
1. `plan_script_credits` 먼저 (플랜 크레딧, 만료됨)
2. `script_credits` 나중 (횟수권 크레딧, 영구)

**환불 조건:**
- 발화 5단어 미만 → `refund_script_credit` RPC
- API 오류 → 자동 환불
