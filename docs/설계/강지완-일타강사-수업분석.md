## 2. OPIc 10가지 유형과 강사 강의의 매핑

### 2-1. OPIc 시험 구조 (정확한 버전)

```
Q1       자기소개 (고정)
─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─
Q2~Q10   일반 콤보 × 3세트 (유동)
         각 콤보: description → routine/comparison → 과거경험
─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─
Q11      roleplay_11 (질문하기)
Q12      roleplay_12 (대안제시 = OPIc 공식 "돌발")
Q13      roleplay_13 (과거경험)
─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─
Q14      advanced_14 (사회적 변화/비교)
Q15      advanced_15 (심화 의견/이슈)

5개 콤보 = 선택형 3개 + 공통형 2개 (고정 비율)
```

### 2-2. 10가지 유형 × 강사 강의 매핑

| question_type_eng | 강사 강의 번호 | 강사가 가르치는 핵심 | 평가 체크박스 |
|-------------------|-------------|------------------|-------------|
| **description** | 4강 (Skeleton Paragraph) | 서론-본론(3섹션)-결론 구조 | F2-05, ADV-2-SP5 |
| **routine** | 4강 (동일 구조 적용) | Skeleton Paragraph + 습관 표현 | F2-05, ADV-2-SP5 |
| **comparison** | 7강 (시제), 14강 (비교변화) | 과거↔현재 대비 + Template A | ADV-1-D2/N2, F1 |
| **past_experience_childhood** | 7강 (시제 Template A) | When I was young, I used to ~ | ADV-1-D2/N2, F1-02/05 |

### 2-3. 강사의 "공통형 주제" 교수 (강사 표현: "돌발")

강사가 10~13강에서 "돌발"이라 부르며 가르치는 것은 실제로 **공통형 주제**에 대한 대응법:

| 강사 분류 | 실제 OPIc 구조 | 주제 예시 |
|----------|-------------|---------|
| "시설 돌발" | 공통형 일반 콤보 (description + routine + 과거경험) | 은행, 호텔, 식당, 교통 |
| "환경 돌발" | 공통형 일반 콤보 | 재활용, 온난화, 지리, 날씨 |
| "산업 돌발" | 공통형 일반 콤보 | 기술, 산업 |
| "개인 돌발" | 공통형 일반 콤보 | 휴일, 모임, 자유시간, 약속 |

**핵심**: 이것들은 별도 유형이 아니라, 서베이에서 선택하지 않아 준비가 안 된 주제의 **일반 콤보**(description + routine + 과거경험)임. 공통형이든 선택형이든 같은 10가지 유형 중 하나로 출제됨.

## 5. 핵심 교수법 분석

### 5-1. Skeleton Paragraph — **모든 답변**의 기본 구조

> "스켈레톤 패러그래프를 쓰면 반드시 IH와 어드밴스의 심사를 받게 된다"

**세부진단서와의 관계:**
- F2-05 (Skeletal paragraphs) = 화자가 **전반적으로** 생산하는 담화 수준
- description만이 아니라 **routine, comparison, 과거경험, 롤플레이, 어드밴스 모두**에서 일관되게 문단 구조 유지 필요
- 강사가 "258부터 연습"이라 한 것은 description이 **가장 쉽게 문단을 만들 수 있는 연습 시작점**이기 때문

**구조:**
```
Topic Sentence (주제 선언)
  → Transition Word (전환어)
    → Supporting Sentence 1
    → Supporting Sentence 2
    → Supporting Sentence 3
  → Concluding Sentence (결론)
```

**콤보 내 description 위치별 변형 (강사의 "258"):**

| 콤보 순서 | 시작 (Topic) | 전환어 | 본론 구분자 | 결론 | 마무리 |
|----------|-------------|-------|-----------|------|--------|
| 콤보① description | To talk about... | To get into more details | first/second/last thing | Overall... | That's about it |
| 콤보② description | Speaking of... | Speaking of that | one/another/last thing | The conclusion is... | That's pretty much about it |
| 콤보③ description | When it comes to... | Speaking of which | good/another good/best thing | The bottom line is... | That's all I can say |

**다른 유형에도 적용:**
- routine: 같은 구조 + 습관/순서 표현
- comparison: 같은 구조 + 과거↔현재 대비
- 과거경험: 같은 구조 + Template A/B
- roleplay_11/12: 도입→본론→마무리 구조
- advanced_14/15: 같은 구조 + 토론 요소

**IH vs AL 차이점:**
- IH: Topic Sentence 간단, Concluding Sentence 간략
- AL: Topic Sentence **가치 판단 문장**, Concluding Sentence **요약+평가**

### 5-2. Template + Slot Filling

> "전부 외우는 건 불가능. 틀만 외워서 체점받도록"

**과거 시제 Template A** (childhood/first 문제):
```
When I was young, I used to [ACTIVITY].
  → Unlike now / Unlike my current [SITUATION],
    [어릴적 디테일]
As for now, [요즘 얘기]
Overall, I still remember... I miss those days.
  I wish I could go back to those moments.
```

**과거 시제 Template B** (계기/변화 문제):
```
The moment I first began [ACTIVITY] was a long time ago.
  → I can't recall the exact moment.
  → However, I do remember the excitement from my childhood.
My parents introduced me to [TOPIC] during my early years,
  that significantly influenced my initial [TOPIC] preferences.
As I grew older, my passion for [TOPIC] grew.
Nowadays, my [TOPIC] taste has developed positively.
```

**14번 만능 (스마트폰):**
```
There are both similarities and differences between the past and the present.
  → In the past, we had to physically visit [PLACE] in person.
  → Which was a waste of time and energy.
However, with technological advancements, this is no longer the case.
  → All you have to do is download an application with just a few clicks.
Perhaps the most significant difference is that we have more options available today,
  all thanks to the invention of smartphones. What a world!
```

**15번 만능 (경제):**
```
People always consider money. It's one of the most valuable assets.
The most significant issue is the recession / inflation,
  caused by [최신 이슈].
  → Prices are increasing. Small businesses are struggling.
In conclusion, we should unite our efforts to overcome this.
```

**공통형 주제 만능 (시설):**
```
[어렵다] It's a tough question. I'll do my best.
[많다]   There are numerous [FACILITIES] in Korea.
[분류]   We refer to [TYPE A] and [TYPE B].
[이용]   A large portion of the population frequents [PLACES] for various purposes.
[요즘]   These days a large number of people use [ONLINE SERVICE] instead of visiting in person.
[자긍심] Foreigners are fascinated by the sophisticated services. I'm proud as a Korean.
[결론]   We are performing well based on the trend so far.
```

```
강사 Phase:              Rule Engine Step:          세부진단서:
─────────────            ─────────────              ─────────────

Phase 5 (14/15)    ←→   Step 6 (AL 판정)      ←→   AL 체크박스 12개
```

**시사점**: 우리 평가 엔진의 체크박스 통과율을 보면, 학생이 어느 Phase에서 막혀 있는지 **자동으로 진단**할 수 있다.

## 9. v3 핵심: 이 강의는 IH→AL 전용이다

### 9-1. 강의 적용 범위의 한계

강의 제목이 **"OPIc AL 강의"**이다. 처음부터 **IH→AL을 목표로 하는 학생** 전용으로 설계되었음.

OPIc 학생은 IL~AL까지 등급과 실력이 모두 다르다. 이 강의의 교수법을 모든 등급에 동일하게 적용하면 안 된다.

### 9-2. 등급별로 필요한 것이 완전히 다르다

평가 엔진 Rule Engine 7-Step의 임계값 기준:

| 목표 등급 | Rule Engine 요구 | 핵심 체크박스 | 필요한 훈련 |
|----------|----------------|-------------|-----------|
| **IM1** | INT 65~75% | INT-1, INT-2-1/2-2, INT-4-F/G | **완전한 문장 1개** 만들기, 10초 이상 말하기 |
| **IM2** | INT 75~95% | INT-2-3, INT-4, ADV-2-SP2~3 | **문장 3~4개 연결** (and, but, so), 간단한 서론/마무리 |
| **IM3** | INT ≥ 95% | INT-3 최대, ADV-2-SP3~4 | **Skeleton 기초**, 질문 6~8개, 시제 기본 정확도 |
| **IH** | INT ≥90% + ADV ≥70% + Q12 통과 | ADV-2-SP5, ADV-4, ADV-1 | **Skeleton 모든 답변**, Q12 대안제시, 시제 50%+ |
| **AL** | IH 조건 + AL ≥70% | AL 12개, ADV-3-V1, ADV-5-G2 | **어휘 업그레이드**, 분사구문, 14/15 전략 |

### 9-3. 강사 교수법의 범용/AL전용 분리

#### ✅ 범용 (모든 등급에 유효)

| 원칙 | 설명 |
|------|------|
| **"틀을 외우고 내용은 자유"** | 등급별로 **다른 수준의 틀** 제공, 암기가 아닌 구조 학습 |
| **"자유 답변 → 교정 → 재발화"** | 교수법 사이클은 등급 무관하게 효과적 |
| **"채점 기준에 맞춰 연습"** | 등급별로 **다른 체크박스** 타겟, 체점 기준 역설계 접근은 유효 |
| **"절대 암기 금지"** | 모든 등급에서 통용 |
| **서베이 묶기 전략** | 시험 범위 최소화는 모든 등급에 유리 |
| **난이도 5-5 전략** | 데이터 기반 — 모든 등급에서 상위 등급 확률 증가 |

### 9-4. 등급별 Template 난이도

강사의 Template은 AL 수준이다. 등급별로 다른 Template이 필요하다:

#### IL~IM1 Template: "1문장 + 이유"
```
I [동사] [목적어].
Because [이유].
That's about it.
```

예: "I like listening to music. Because it makes me comfortable. That's about it."

#### IM2 Template: "서론 + 2~3문장 + 마무리"
```
I'd like to talk about [주제].
[문장 1]. And [문장 2]. Also, [문장 3].
That's about it.
```

예: "I'd like to talk about my house. I live in an apartment. And it has three bedrooms. Also, my favorite room is my bedroom. That's about it."

#### IM3 Template: "서론 + 2 Section + 마무리"
```
[주제를 말하겠습니다].
First, [내용 1]. [상세 1문장].
Also, [내용 2]. [상세 1문장].
Overall, [마무리 1문장]. That's about it.
```

#### IH Template: 강사의 Skeleton Paragraph 그대로
```
To talk about [주제], [Topic Sentence].
To get into more details, the first thing is that [1]. [상세].
The second thing is that [2]. [상세].
And the last thing is that [3]. [상세].
Overall, [결론 요약]. That's about it.
```

#### AL Template: 강사의 강화 버전
```
To talk about [주제], [가치 판단 Topic Sentence]. [추가 1문장].
To get into more details, the first thing is that [1]. [상세 + 분사구문].
The second thing is that [2]. [상세 + 관계절].
And the last thing is that [3]. [상세 + 고급 어휘].
Overall/The bottom line is that [결론 요약 + 평가]. That's about it.
```

### 9-7. 최종 결론: 현재 구현의 보강 자료

> **현재 구현(C→D→QSE→E→L1→F 파이프라인, 병목 기반 개인화)의 방향이 맞다.**
>
> 강지완 강의에서 추출할 것은 "특정 표현/전략"이나 "등급별 고정 경로"가 아니라,
> **현재 파이프라인에 빠져 있는 판정 영역을 보강하는 데 쓸 재료**이다:
>
> 1. **Layer 1에 Skeleton Paragraph 구조 판정 추가** — IM↔IH 분류의 핵심
> 2. **Layer 1에 어휘 판정 추가** — IH↔AL 차이의 핵심 (현재 전혀 없음)
> 3. **Layer 1에 문법 판정 추가** — AL 감점 항목 (Agreement, Preposition, Syntax)
> 4. **Prompt C에 체크박스 통과율 추가** — 진단 정확도 보강
> 5. **Prompt E에 등급별 난이도 지시 추가** — 저등급 과부하 방지
> 6. **Layer 2에 AI 교정 비교법 추가** — AL 목표 학생 전용
>
> 이것이 "강사를 AI로 복제"하는 것이 아니라, **"강사의 관찰을 기존 시스템의 빈 곳에 채우는 것"**이다.
