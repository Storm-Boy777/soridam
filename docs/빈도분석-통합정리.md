# 빈도 분석 통합 정리

> **데이터 소스**: 소리담 235건 시험 후기 (일반 532회 + 롤플레이 161회 + 어드밴스 132회 = 총 825회)
> **목적**: 전략 페이지 섹션 5(선택형), 섹션 6(공통형), 커버리지 계산 재정리

---

## 핵심 프레임워크

```
OPIc 15문제 = 5콤보
├── 선택형 3콤보 (60%) → 내가 선택한 주제 → 전부 학습 (빈도 = 학습 순서)
└── 공통형 2콤보 (40%) → 통제 불가 → 빈도 상위 N개 학습 (빈도 = 추가 커버리지)

최종 커버리지 = 60% + (40% × 공통형 상위10개 커버율 59.7%) = 60% + 24% = 84%
```

## 4. 최종 커버리지 계산

```
선택형 60% (전부 학습)
+ 공통형 40% × 상위 10개 커버율 59.7%
= 60% + 24%
= 84%
```

| 구분 | 비중 | 학습 범위 | 커버리지 |
|---|---|---|---|
| 선택형 3콤보 | 60% | 9개 주제 전부 학습 | **60%** |
| 공통형 2콤보 | 40% | 상위 10개 주제 학습 | **24%** (40% × 59.7%) |
| **합계** | 100% | **19개 주제** | **84%** |

> **결론**: 선택형 9개 + 공통형 상위 10개 = **총 19개 주제**만 학습하면
> 시험의 **84%를 사전에 대비**한 상태로 응시할 수 있다.

## 5. 전략 페이지 적용 시 변경사항

### 섹션 5 (선택형 빈도)
- 제목: "선택형 3콤보 = 60%는 이미 내 것"
- 테이블: 위 선택형 9개 (합계 기준 정렬, 분류: 핵심/우선)
- 포인트: "전부 학습하되, 이 순서로 우선순위를 정한다"

### 섹션 6 (공통형 빈도)
- 제목: "공통형 2콤보 — 상위 10개로 24% 추가"
- 테이블: 위 공통형 상위 10개 (합계 + 누적 커버리지)
- 포인트: "10개만 학습하면 공통형의 60%를 커버"

### 최종 커버리지 섹션
- 60% (선택형) + 24% (공통형 상위 10개) = **84%**
- 현재 페이지의 "90%+" → **"84%"**로 수정 (더 정확한 수치)

## 6. 검토 필요 사항

### 매핑이 애매한 항목
- **가족 집 돌봐주기 (9회)** → "집"으로 매핑. "집에서 보내는 휴가"일 수도?
- **친구 만남 계획 (10회)** → "모임"으로 매핑. "약속"일 수도?
- **공연/콘서트 (17회)** → 일반 카테고리에 없고 롤플레이에만 존재. 선택형 맞는지?

### 수치 관련
- 선택형 비중 60% vs 실제 데이터 56.7% (468/825) → 60%는 "3콤보/5콤보" 이론값, 유지?
- 현재 "90%+" → "84%"로 변경 시 임팩트 감소? 또는 "80%+"로 표현?
